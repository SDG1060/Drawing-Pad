Welcome

Please accept my warm welcome to my software. This guide will show you how to use the Drawing Pad app. So, let's get started.

Drawing Pad is an advanced position and art making software based on Javascript. These app is perfect for those who haven't have keen knowledge about screen window positioning. The screen window positioning means the position of each of point of the computer window. This topic is very important for basically Front end web development and if you want to make cool design, This app is perfect for you. So, let's jump to the tutorials.

If you just start the app, you will see a blank page and there is only one menu button. You can Click the button or can press 'n' to open the navigation menu (the 'n' key press is the shortcut of clicking the navigation button).

Then, if you click it or press 'n', you will see a good looking template like this.
I will explain it to you frame by frame.

  • If you press the Question Mark icon, you will find this page.
  • The position Button is a default button. If you click This, you will find that the top of the background vanished. If you want it again, you have to simply click that position button again.
  • The line width input box shows us the line width of our pen. If you want a thicker pen, you simply have to type or increase it.
    Note: You can't decrease the value of line width to 0 or less.
  • The forth button is Color Swatch. In this swatch, you can select the color of your pen. The default value is black.

Coping the position:

If you press 'c' button on the keyboard, you can copy the current position of your mouse. And thus you can easily know where is your mouse. This feature helps you to design a page.

Drawing:

If you have selected your line width and color, you simply drag to a point and draw whatever you want.
